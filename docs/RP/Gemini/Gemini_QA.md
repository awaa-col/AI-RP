# Gemini 内容重复？
- 模型底层问题，暂时无法根治，等Google更新- 目前最好的办法是从第一次重复开始就改输入内容重刷到不重复，否则重复容易病毒式蔓延
# 1.5Pro 和Flash 哪个好？
- 总体来说Pro更好，Flash速度更快，智力均优，上下文Pro两倍于Flash，Pro的输出更
精简，Flash的安全过滤更低
# Pro免费版&Flash免费版？
- pro 每分钟两条，每分钟32ktokens输入，(Gemini的词符计数与酒馆不同，32k等同于
酒馆的60k)目前无限制RPD。- flash 几乎无限制。
# Pro什么时候收费？

# 截断过于频繁？
