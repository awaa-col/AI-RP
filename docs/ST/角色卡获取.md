- 世界书(部分结构较为简单的卡没有)

- 提示"External media haas been blocked"怎么办

        - ![alt text](image-1.png)
- 提示"clould not import characters"
    - 这是由于您下载错角色卡导致的
        - 请保证您下载的角色卡是角色卡而不是封面,部分作者会将封面(无法使用)作为首楼图片,你需要保存正确的角色卡
